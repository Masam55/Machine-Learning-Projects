# Machine Learning Projects Portfolio

This repository contains a collection of machine learning projects built using Python and Jupyter Notebook. Each project demonstrates core concepts in data analysis, statistical modeling, and predictive analytics.

## 🔧 Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Jupyter Notebook

## 📊 Projects

### 1. Polynomial Regression
- Built a polynomial regression model to capture non-linear relationships
- Performed feature transformation and model evaluation
- Demonstrates handling of non-linear data patterns

### 2. Simple Linear Regression
- Developed a basic regression model to predict outcomes
- Performed data visualization and model evaluation
- Demonstrates foundational machine learning concepts

### 3. Multiple Linear Regression
- Built a multi-variable regression model
- Analyzed relationships between multiple features and target variable
- Demonstrates real-world predictive modeling use case

## 📌 Key Skills Demonstrated
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Statistical Modeling
- Machine Learning Algorithms
- Data Visualization
- Model Evaluation
